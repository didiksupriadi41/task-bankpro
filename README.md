# Bank Pro

#### Deskripsi aplikasi

Bank Pro adalah Aplikasi web bank sederhana yang digunakan oleh nasabah
Bank Pro.
Aplikasi ini diimplementasikan diatas menggunakan
[**ReactJS**](https://reactjs.org/)
